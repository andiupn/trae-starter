# Running Tests

## When
Ketika user ingin menjalankan test suite proyek.

## How
1. Identifikasi framework testing proyek
2. Jalankan command test sesuai proyek
3. Laporkan hasil test

## What
- Status test (berhasil/gagal)
- Detail error jika ada
