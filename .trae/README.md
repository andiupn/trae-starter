# TRAE Starter Configuration
Direktori ini berisi konfigurasi TRAE IDE DASAR untuk pemula dan proyek baru!

## Struktur Lengkap
```
.trae/
├── README.md
├── mcp.json.template       # Template MCP!
├── rules/
│   ├── main-rules.md
│   ├── output-formatting.md
│   └── web-dev/            # Rules web dev sederhana
└── memories/               # Project memories!
    └── README.md
```

## Best Practices
Sudah termasuk:
- Rules dengan metadata
- Memories untuk konteks jangka panjang
- Template MCP (maks 40 tools!)
- Direktori skills di .agents/skills/
- Struktur sengaja minimal agar mudah dipakai sebagai template awal

## Cara Menggunakan
- Clone repo sebagai template
- Atur sesuai kebutuhan
- Mulai chat baru di TRAE IDE!
