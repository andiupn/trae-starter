# Contributing to this Template

Thank you for your interest in contributing! 🎉

---

## 🐛 Reporting Bugs
Use our Bug Report Template in `.github/ISSUE_TEMPLATE/bug_report.md`:
1. Check existing Issues.
2. Search for similar reports.
3. If not found, create a new issue.
4. Include clear reproduction steps, expected/actual behavior, and environment details.

---

## 💡 Suggesting Features
Use our Feature Request Template in `.github/ISSUE_TEMPLATE/feature_request.md`:
1. Describe the problem/use case.
2. Propose your solution.
3. Explain who would benefit.

---

## 📐 Code Style Guidelines
- Indentation: 2 spaces.
- Language: Bahasa Indonesia for communication, English for code.
- Code conventions: Standard best practices for the respective tech stack.
