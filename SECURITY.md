# Security Policy

Please help us keep this project secure. We take security issues seriously and appreciate your efforts to report them responsibly.

---

## 🛡️ Supported Versions

Only the latest release of this template/workspace is actively supported.

| Version | Supported |
| ------- | --------- |
| Latest (main branch) | ✅ Yes |
| All Older Versions | ❌ No |

---

## 📞 Reporting a Vulnerability

**Do not open public GitHub Issues for security vulnerabilities.**

If you discover a security vulnerability (such as exposed credentials, security holes, or execution exploits in scripts), please report it directly through:

👉 **[Contact Form at kuncimu.com](https://kuncimu.com)**

We will acknowledge your report, investigate the issue, and provide a fix as quickly as possible. Thank you for helping us maintain a safe community.

---

# Kebijakan Keamanan (Bahasa Indonesia)

Bantu kami menjaga keamanan proyek ini. Kami menangani masalah keamanan dengan serius dan sangat menghargai kontribusi Anda untuk melaporkannya secara bertanggung jawab.

## 🛡️ Versi yang Didukung

Hanya versi rilis terbaru pada cabang `main` yang didukung secara aktif.

## 📞 Melaporkan Kerentanan

**Jangan membuat GitHub Issues publik untuk kerentanan keamanan.**

Jika Anda menemukan kerentanan keamanan (seperti kredensial yang terekspos, celah keamanan, atau eksploitasi eksekusi pada skrip), harap laporkan langsung melalui:

👉 **[Formulir Kontak di kuncimu.com](https://kuncimu.com)**
