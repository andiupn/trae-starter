Silakan simpan captura de pantalla del sitio web TRAE Starter di carpeta ini!

## Archivo Daftar yang Dibutuhkan
- `homepage.png`: Captura de pantalla de la página de inicio de tampilan utama
- (opcional) `mobile-view.png`: Tampilan di perangkat mobile
- (opcional) `demo-feature.png`: Demostración interactiva

## Cara Menjalankan Web Sample para captura de pantalla
Utilice Docker para mostrar la muestra web:

```bash
# Di folder trae-starter
docker run -d -p 9002:80 --name trae-starter-web -v $(pwd)/public:/usr/share/nginx/html nginx:alpine

# Buka di browser: http://localhost:9002
```
